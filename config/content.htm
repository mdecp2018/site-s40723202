課程介紹

機械設計系-計算機程式，我們要在Week1 to Week 9 將老師發派的作業完成，首先到github用學校email申請帳號，以 s為首加上學號作為帳號例如s40723202，之後進入課程網站mde.tw領取作業。

Week 1

首先建立github帳號，新增倉儲: s40723202.github.io

接著用git clone指令將網址clone下來：

git clone https://github.com/mdecourse/2018fall

git clone https://github.com/s40723210/s40723210.github.io

再從2018fall資料夾複製裡面的檔案，排除.git和README.md，貼到自己的資料夾內

圖片可按滑鼠右鍵，新增新分頁，即可放大。

接著使用下列指令

1. git status 檢查狀態

2. git add .新增

3. git commit -m "名稱" 提交

4. git push 推送

上傳遠端完成後，就會再https://github.com/mdecp2018/site-s40723202看到上傳的檔案

老師示範影片

Week 2

領取個人作業

登入github帳號後，到https://classroom.github.com/a/wlLbt5TQ領取個人倉儲作業

接著前往site-s40723202 settings下方的Branches 將branches改為gh-pages

Week 3

更改網頁標題

使用應用程式cmd 中到Y:\tmp\site-s40723202

後方輸入python wsgi.py 按 Enter 開啟動態

進入login 輸入密碼:admin，(進入後可至Config中更改)

點選Config ，更改網頁標題Site Title:

更改完畢後按下send，再按下網站右上方generate_pages 按紐推送至近端(資料夾)

根據上周推送之過程 git status > git add . > git commit -m "名稱" > git push完成推送

Week 4

更改reveal投影片

再資料夾p37中啟動Start將會自動開啟Leo Editor

用Leo找到Y:\tmp\site-s40723202\config 開啟reveal.leo檔案

找到封面投影片分支"增加 ###40723202林姵綺個人投影片"

Leo Editor按下Save

使用SciTE文字編輯器中開啟靜態http-server. py可直接再近端檢查更正內容

再推上遠端 git status > git add . > git commit -m "名稱" > git push完成推送
